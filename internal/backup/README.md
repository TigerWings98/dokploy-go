# backup

S3 兼容存储的数据库备份和恢复，使用 RClone CLI 执行。
支持定时 Cron 备份和手动触发，覆盖 5 种数据库类型的 dump/restore 命令生成。

## 资产清单

| 文件 | 输入/输出 | 核心逻辑 |
|------|----------|----------|
| backup.go | 输入: 备份配置 (数据库类型 + 目的地) / 输出: 备份文件路径 | 数据库备份：生成 dump 命令 (pg_dump/mysqldump/mongodump 等) → 执行 → RClone 上传到 S3 |
| restore.go | 输入: 备份文件路径 + 数据库配置 / 输出: 恢复结果 | 数据库恢复：RClone 下载 → 解压 → 执行 restore 命令；备份文件列表查询 |
