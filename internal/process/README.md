# process

本地和远程 SSH 命令执行的统一抽象层。
几乎所有 Docker/构建/Git/备份操作都通过此模块执行，是系统的基础设施底座。

## 资产清单

| 文件 | 输入/输出 | 核心逻辑 |
|------|----------|----------|
| exec.go | 输入: 命令参数 + 可选回调 / 输出: 执行结果或实时输出流 | 本地命令执行：ExecAsync（同步获取输出）、SpawnAsync（实时流式回调） |
| ssh.go | 输入: SSH 连接配置 + 命令 / 输出: 远程执行结果 | SSH 远程执行：建立 SSH 连接 → 执行命令 → 返回输出，支持密钥认证 |

> 自指声明：一旦本目录下的逻辑发生变化，必须立即同步更新本 README。
