# traefik

Traefik v3 反向代理配置生成和域名路由管理。
通过 YAML 文件提供者动态管理路由，每个应用一个独立配置文件，格式与原 TS 版完全兼容。

## 资产清单

| 文件 | 输入/输出 | 核心逻辑 |
|------|----------|----------|
| config.go | 输入: 无 / 输出: Traefik YAML 类型定义 | Traefik 路由/服务/中间件的 Go struct 定义（含 AddPrefix 中间件） |
| manager.go | 输入: 域名/应用配置 / 输出: YAML 配置文件 | 域名路由管理：创建/更新/删除 Traefik 配置文件、中间件配置、证书处理 |

> 自指声明：一旦本目录下的逻辑发生变化，必须立即同步更新本 README。
